## <span style="color: #ff9900; border-bottom: 2px solid #ff9900; padding-bottom: 5px;">🚀 Getting Started</span>

```bash
# Clone the repository
git clone https://github.com/xorascs/azamon.git

# Install dependencies
npm install

# Start the /frontend
npm run dev

# Start the /backend
npm run start:dev
```
